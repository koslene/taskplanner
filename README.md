# Mācību uzdevumu plānotājs

Vienkārša Python programma ar GUI, kas palīdz pārvaldīt mācību uzdevumus.

## Funkcijas
- Uzdevumu pievienošana
- Uzdevumu rediģēšana
- Uzdevumu dzēšana
- Statusa maiņa
- Meklēšana
- Filtrēšana pēc statusa
- Datu saglabāšana JSON failā

## Tehnoloģijas
- Python
- Tkinter
- JSON

## Palaišana
1. Instalē Python 3
2. Saglabā failu kā `task_planner.py`
3. Palaiž:
   ```bash
   python task_planner.py
